# 🎮 ผจญภัยอัลกอริทึมในชีวิตประจำวัน

## Everyday Algorithm Adventure

> เกมการศึกษา (Educational Web Game) สำหรับเด็ก เพื่อเรียนรู้แนวคิดเชิงคำนวณ (Computational Thinking) และการออกแบบอัลกอริทึมผ่านสถานการณ์ในชีวิตประจำวัน

---

## 📖 ภาพรวมโครงการ

**Everyday Algorithm Adventure** เป็นเกม Web App แบบ Single-file ที่พัฒนาด้วย HTML, CSS และ JavaScript ภายในไฟล์เดียว โดยออกแบบมาเพื่อช่วยให้เด็กอายุ 6–12 ปี เรียนรู้แนวคิดพื้นฐานของอัลกอริทึม (Algorithmic Thinking) ผ่านการเรียงลำดับเหตุการณ์ในชีวิตประจำวันอย่างถูกต้อง

ผู้เล่นจะได้ผจญภัยผ่านโลกต่าง ๆ ตั้งแต่บ้าน โรงเรียน ชุมชน เมืองไฮเทค จนถึงปราสาทเวทมนตร์ พร้อมสะสมเหรียญ ดาว และพัฒนาทักษะการคิดเชิงลำดับขั้นตอน

---

# 🎯 วัตถุประสงค์การเรียนรู้

เกมนี้ออกแบบเพื่อส่งเสริม

* Sequence Thinking (การคิดเป็นลำดับขั้นตอน)
* Computational Thinking (แนวคิดเชิงคำนวณ)
* Problem Solving Skills (การแก้ปัญหา)
* Logical Reasoning (การใช้เหตุผลเชิงตรรกะ)
* Everyday Algorithm Awareness (การมองเห็นอัลกอริทึมในชีวิตประจำวัน)

## 🗺️ ระบบแผนที่การผจญภัย

จำนวนทั้งหมด

**15 ด่าน**

แบ่งเป็น 5 โลก

| โลก | ชื่อโลก                | ด่าน  |
| --- | ---------------------- | ----- |
| 🏡  | บ้านแสนอบอุ่น          | 1–3   |
| 🏫  | โรงเรียนสุดหรรษา       | 4–6   |
| 🌳  | ชุมชนน่าอยู่           | 7–9   |
| 🏙️ | เมืองใหญ่ไฮเทค         | 10–12 |
| 🏰  | ปราสาทเวทมนตร์แห่งโค้ด | 13–15 |

สถานะด่าน

* 🔓 ปลดล็อกแล้ว
* ⭐ ผ่านแล้ว
* 🔒 ยังไม่ปลดล็อก

## 🎮 ระบบการเล่น

ผู้เล่นจะได้รับชุดของเหตุการณ์ที่ถูกสลับลำดับ

ตัวอย่าง

แปรงฟันตอนเช้า

1. หยิบแปรงและบีบยาสีฟัน
2. ใช้แปรงขัดถูฟัน
3. บ้วนปากด้วยน้ำสะอาด

ผู้เล่นต้องเรียงลำดับให้ถูกต้อง

### ขั้นตอนการเล่น

1. แตะตัวเลือก
2. ย้ายเข้าสู่พื้นที่คำตอบ
3. จัดเรียงลำดับจากบนลงล่าง
4. กดตรวจสอบคำตอบ
5. รับผลลัพธ์ทันที
